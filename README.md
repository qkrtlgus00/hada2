# 하다 (hada2) — 주간 스케줄 데스크톱 앱

교사용 스케줄 대시보드 스타일의 **주간 캘린더 일정 관리** 데스크톱 앱입니다.
왼쪽 사이드바 + 요일×시간 그리드에 일정을 색깔 블록으로 보여줍니다.
Electron 기반이라 Windows · macOS · Linux 어디서나 실행됩니다.

## 주요 기능 (멀티 대시보드)

- 🏠 **홈** — 오늘 일정·남은 할일·다가오는 마감·이달 지출·습관을 한눈에 (카드 클릭 시 이동)
- ◷ **일정** — 왼쪽에 미니 월간 달력 + **할일 체크리스트** + **반복 목록**(매일/평일/매주/매월),
  오른쪽에 큰 주간 캘린더 · 일정 블록은 **꾹 눌러 드래그로 시간/요일 이동**(15분 스냅)
- ✅ **날짜별 할일** — 자정이 지나면 새 목록으로 시작, **미완료 할일은 다음날로 자동 이동**,
  미니 달력에서 날짜를 누르면 그날 할일 표시(‘오늘’ 버튼으로 복귀)
- 🎨 **Google Drive 스타일 UI** — 파란 패턴 사이드바 + 흰 라운드 본문 + 퀵액세스 카드 + 깔끔한 표(호버 강조)
  · 홈 빠른 액세스 카드는 **한 번 누르면 선택, 다시 누르면 진입**
- 💰 **가계부** — 수입/지출 기록, 이달 요약(수입·지출·잔액)
- 🗂️ **작업 관리** (마감·커미션·외주 통합) — 제목·클라이언트·연락처·플랫폼·유형·금액·상태(대기/진행중/완료)·마감·메모를
  한 카드에 담아 관리. **카드를 눌러 메모 등 항목 수정**, D-day 정렬·임박/지남 강조, 항목별 진행률(%) 슬라이더, 상단 상태별 건수·금액 요약
- 🔁 **반복 목록 날짜 기록** — 반복 항목의 이름/규칙 또는 **📅 버튼**을 눌러 뜬 **미니 달력**에서 원하는 날짜를 골라 완료 표시
- 📝 **메모** — 카드형 메모(자동 저장)
- ✅ **습관** — 최근 7일 체크 + 연속(streak) 기록
- ▶ **음악(유튜브)** — 유튜브 링크를 붙여넣어 백그라운드 오디오로 재생 (인터넷 필요)
  · 음량 슬라이더 · **광고/다른 곡 소리 차단**(로드 즉시 음소거 후 지정 영상만 확인되면 재생, 광고는 스킵/빨리감기)
- 🔔 **일정·마감 알림** — 일정별 알림(정각~1일 전) + 작업 마감 자동 알림(전날·당일 09:00)
- 🗕 **트레이 최소화** — 창을 닫아도 트레이에 상주하며 음악 계속 재생, 트레이 메뉴에서 열기/재생·일시정지/다음 곡/종료
- 미니멀 **라인 아이콘** UI (이모지 없음)
- 🎨 **사용자 지정** — 앱 제목, 강조색·사이드바색(프리셋 포함), 라이트/다크
- 🪟 **커스텀 타이틀바** — OS 창틀/영어 메뉴바 없이 앱과 한 몸인 상단바 (최소화/최대화/닫기)
- ✨ **투명 효과 토글·화면 배율** — 기본은 **완전 불투명(솔리드)**, 설정에서 '투명 효과'를 켜면 창이 실제로 투명해져 바탕화면이 비침(재시작 없이 즉시 전환). 글자는 그림자/불투명 카드로 항상 또렷
- 🖼️ **스티커 꾸미기** — 뷰마다 이미지 스티커를 올리고 **드래그 이동 · 모서리로 크기/회전 · 투명도 · 앞뒤 순서 · 잠금(고정)**
- 💾 **데이터 백업** — 전체 데이터 JSON 내보내기/가져오기
- 🔄 **자동 업데이트** — 소스 실행 시 시작할 때 안정 브랜치(`main`)에서 최신 파일을 받아 적용
- 로컬 저장 (외부 서버 없음)

## (구) 기능 목록

- 🗓️ **주간 캘린더** — 월~일 × 06:00–22:00 시간 그리드
- ➕ **일정 추가/수정/삭제** — `+ Add Event` 버튼 또는 빈 칸 클릭
- 🎨 **카테고리 색상** — Math·Art·Physics·Sport 등 카테고리별 파스텔 색
- 🔎 **카테고리 필터 & 검색** — 상단 칩으로 필터, 🔍로 제목·메모 검색
- ✅ **Confirmed 배지** — 확정된 일정 표시
- 📍 **오늘 열 강조 + 현재 시각 라인** — 지금 시간이 빨간 선으로 표시
- ◀▶ **주 이동** — 이전/다음 주, "This week"로 이번 주 복귀
- ⏰ **오늘 일정 알림** — 시작 시 네이티브 알림
- 🌙 **라이트/다크 모드**
- 🎨 **색상 사용자 지정** — 설정에서 **강조색**과 **사이드바 배경색**을 각각 지정
  (프리셋 + 색상 선택기, 텍스트 대비 자동)
- 📁 **파일 화면** — 내 컴퓨터 폴더를 골라 파일을 표/카드로 보기, 폴더 이동·파일 열기
- 💾 **로컬 저장** — 데이터는 내 컴퓨터에만 저장 (외부 서버 없음)

> 사이드바의 Overview·Exams·Students 등 다른 메뉴는 **디자인용 표시**이며,
> 실제 동작은 Schedule(캘린더) 화면입니다.

## 실행 방법

Node.js(18 이상)가 설치돼 있어야 합니다.

```bash
npm install     # 의존성(electron) 설치
npm start       # 앱 실행
```

> Windows PowerShell에서 `npm` 실행이 막히면 `npm.cmd install` / `npm.cmd start`
> 로 실행하거나, cmd 창에서 실행하세요.

## 데이터 저장 위치

일정 데이터는 OS별 사용자 데이터 폴더의 `data.json`에 저장됩니다.

- **Windows**: `%APPDATA%/hada2/data.json`
- **macOS**: `~/Library/Application Support/hada2/data.json`
- **Linux**: `~/.config/hada2/data.json`

저장은 임시 파일에 쓴 뒤 교체하는 **원자적 방식**이라 안전합니다. 이전 버전의
할 일(목록형) 데이터가 있으면 자동으로 일정으로 변환됩니다.

## 프로젝트 구조

```
package.json      # electron 의존성, start 스크립트
src/
  main.js         # 메인 프로세스: 창 생성, 파일 저장/로드, 알림
  preload.js      # contextBridge 로 안전한 API(window.api) 노출
  index.html      # 앱 셸(사이드바 + 캘린더) 마크업
  renderer.js     # 캘린더 로직 (주간 날짜, 이벤트 CRUD, 그리드 렌더, 필터, 모달)
  styles.css      # 스타일 (라이트/다크)
```

## 데이터 모델

```js
Event {
  id, title,
  date: "2025-01-02",   // 날짜
  start: "07:00",       // 시작 시간
  end: "07:40",         // 종료 시간
  category: "Math",     // 카테고리(색상)
  confirmed: false,     // Confirmed 배지
  notes: "",            // 메모
  remindMin: 10,        // 알림(분 전). null/없음 = 알림 없음
  createdAt, updatedAt
}
```

## 보안

- `contextIsolation: true`, `nodeIntegration: false`, `sandbox: true`
- 렌더러는 preload가 노출한 `load / save / notify` 함수 외 시스템 접근 불가
- CSP(Content-Security-Policy)로 외부 리소스 로드 차단

## 향후 확장 아이디어

일정 겹침(중복 시간) 나란히 배치, 드래그로 시간 이동, 월간 뷰, 반복 일정,
데이터 내보내기(JSON/CSV/ICS), 사이드바 메뉴 실제 기능화
